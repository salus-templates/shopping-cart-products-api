# shopping-cart-products-api

## Overview

The Products API, written in Dotnet, is part of the Shopping Cart experience. It provides the product catalog as a Restful API.

## Environment Variables

- ConnectionStrings__ApplicationDbConnection - The database connection string. For example; User ID=shoppingcartadmin;Password=mysecurepassowrd;Server=127.0.0.1;Port=5432;Database=shoppingcart;Pooling=true;
- ASPNETCORE_URLS - The bind address for the application; 0.0.0.0:8080
